# Project scope — Methodology v0.7

This is an independent public-research workspace for a Global and Thailand Logistics Intelligence Dashboard.

The system may use only public sources, repository materials, and methodology formally recorded in this repository. It must not use saved personal memories, conversations outside the project, employment information, company information, or inferred organizational context.

The dashboard monitors external logistics situations, assesses potential effects, presents conditional preparedness options, and reviews innovations. It does not provide organization-specific instructions or legal advice.
